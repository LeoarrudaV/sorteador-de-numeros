# 🎲 Sorteador de Números

Um projeto simples desenvolvido em **HTML, CSS e JavaScript puro**, com foco em praticar manipulação de DOM, lógica de programação e validação de dados.

---

## 🚀 Funcionalidades

* Definir a **quantidade** de números a serem sorteados
* Definir o **intervalo inicial (de)**
* Definir o **intervalo final (até)**
* Impedir números repetidos
* Validação para evitar:

  * Intervalo inválido (`de` maior que `até`)
  * Quantidade maior que o total disponível no intervalo
* Alteração dinâmica do botão **Reiniciar**
* Atualização dinâmica do resultado na tela

---

## 🧠 Conceitos praticados

* Manipulação do DOM (`document.getElementById`)
* Uso de `value` para capturar dados do usuário
* Conversão de tipos com `parseInt()`
* Estruturas de repetição (`for` e `while`)
* Estruturas condicionais (`if`)
* Arrays e método `push()`
* Método `includes()` para evitar repetição
* Template Strings (`` `${variavel}` ``)
* Manipulação de classes CSS com `classList`
* Uso de `innerHTML` para atualizar conteúdo
* Validação de entrada de dados
* Prevenção de loop infinito

---

## 📸 Preview
